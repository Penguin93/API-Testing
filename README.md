# API-Testing
# Using API functions
This program utilises API functions to retrieve data from a URL and return data to the user, filtering out all unwanted data. The current function of this program is to retrieve information of pokemon and return specific information about the pokemon. This process can work with any API depending on the information the user wishes to seek.

# Requirements
To run this program, you must install the following dependencies:
- `requests` to make HTTP requests to the API
